Kebijakan Privasi

ADK PHOTOCOPY · Terakhir diperbarui: Juni 2026

Kebijakan Privasi ini menjelaskan bagaimana ADK PHOTOCOPY (adkivia.com) mengumpulkan, menggunakan, dan melindungi informasi yang kamu berikan saat menggunakan layanan kami. Dengan menggunakan layanan kami, kamu menyetujui kebijakan ini.

1. Informasi yang Kami Kumpulkan

Kami hanya mengumpulkan informasi yang diperlukan untuk menjalankan layanan:

  • Nomor telepon atau email — untuk verifikasi identitas via OTP dan mengakses riwayat pesanan
  • File dokumen — yang kamu upload untuk diproses sesuai layanan yang dipilih
  • Data transaksi — nomor order, jumlah pembayaran, metode, dan status pembayaran
  • Data teknis — log aktivitas sistem untuk keperluan debugging dan peningkatan layanan

Kami tidak meminta nama lengkap, alamat, NIK, atau data sensitif lainnya. Nomor telepon atau email sudah cukup untuk menggunakan layanan kami.

2. Cara Kami Menggunakan Informasi

Informasi yang dikumpulkan digunakan semata-mata untuk:

  • Memproses dan menyelesaikan pesanan kamu
  • Mengirim notifikasi status pesanan melalui WhatsApp
  • Memverifikasi kepemilikan pesanan via OTP agar hanya kamu yang dapat mengakses hasil
  • Menganalisis pola penggunaan untuk meningkatkan kualitas layanan
  • Menangani pertanyaan dan pengaduan dari pengguna

3. Penyimpanan File Dokumen

  • File dokumen yang kamu upload akan otomatis dihapus dari server dalam 30 hari sejak tanggal upload
  • File tidak disimpan permanen dan tidak digunakan untuk tujuan lain selain memproses pesananmu
  • Setelah 30 hari, file tidak dapat diunduh lagi — segera simpan file hasil ke perangkatmu
  • Data transaksi disimpan lebih lama untuk keperluan audit dan laporan keuangan

4. Keamanan Data

  • Seluruh komunikasi antara browser kamu dan server kami dilindungi dengan enkripsi HTTPS/TLS
  • Informasi pembayaran diproses oleh TriPay selaku penyedia layanan pembayaran berlisensi — kami tidak menyimpan data kartu atau rekening bank kamu
  • Akses ke hasil pesanan hanya bisa dilakukan oleh nomor telepon/email yang terdaftar pada pesanan tersebut
  • Sistem OTP digunakan untuk mencegah akses tidak sah ke riwayat pesananmu

5. Berbagi Data ke Pihak Ketiga

Kami tidak menjual, menyewakan, atau membagikan data pribadimu untuk tujuan komersial. Data hanya diteruskan ke mitra layanan berikut, sesuai fungsinya:

  • TriPay — untuk memproses pembayaran pesanan
  • Fonnte — untuk mengirim notifikasi WhatsApp terkait status pesanan

Kedua mitra di atas hanya menerima data yang diperlukan dan terikat kewajiban kerahasiaan.

6. Hak Kamu sebagai Pengguna

  • Akses — kamu dapat melihat riwayat pesanan kapan saja menggunakan nomor telepon/email yang terdaftar
  • Penghapusan — kamu dapat meminta penghapusan data dengan menghubungi kami; akan diproses dalam 7 hari kerja
  • Koreksi — jika ada data yang tidak akurat, hubungi kami melalui WhatsApp untuk perbaikan

7. Cookie

Website kami menggunakan cookie terbatas hanya untuk:

  • Menyimpan preferensi tampilan (mode terang/gelap)
  • Menjaga sesi sementara selama menggunakan layanan

Kami tidak menggunakan cookie untuk pelacakan iklan atau analytics pihak ketiga.

8. Perubahan Kebijakan Privasi

Kebijakan ini dapat diperbarui sewaktu-waktu. Tanggal pembaruan terbaru selalu tercantum di bagian atas halaman ini. Penggunaan layanan setelah perubahan diterbitkan dianggap sebagai persetujuan atas perubahan tersebut.
